## Análisis de una página web

- Una web se conforma de 3 secciones que son:

- Header
- Main
- Footer

Dentro del header se puede usar la etiqueta section, que va a ayudar a encontrar o a diferenciar cada parte de la página

Dentro del main va la etiqueta article en donde se puede combinar con la etiqueta id para definir cada sección de la página.
