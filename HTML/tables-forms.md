# Guía Esencial: Formularios y Tablas en HTML

## 1. Tablas (`<table>`)
**Concepto:** Se utilizan para organizar y mostrar datos de manera tabular (filas y columnas). No deben usarse para diseñar el diseño (layout) de la página, solo para datos.

**Estructura básica:**
- `<table>`: Contenedor principal.
- `<tr>`: Una fila (Table Row).
- `<th>`: Encabezado de celda (Table Header), el texto sale en negrita por defecto.
- `<td>`: Celda de datos (Table Data).

**Ejemplo de aplicación:** Listado de usuarios, inventarios o reportes financieros.

---

## 2. Formularios (`<form>`)
**Concepto:** Son la herramienta principal para recolectar información del usuario y enviarla a un servidor.

**Componentes clave:**
- `<form>`: Define el inicio y fin del formulario.
- `<label>`: Define una etiqueta para un elemento de entrada (mejora la accesibilidad).
- `<input>`: El elemento más versátil; puede ser texto, contraseña, checkbox, etc.
- `<button>`: Ejecuta la acción de envío o reinicio.

**Ejemplo de aplicación:** Pantallas de inicio de sesión, registros de contacto o encuestas.
